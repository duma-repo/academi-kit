### 安装依赖

python 版本需要 3.10 或以上

执行下面命令安装依赖包
```shell
pip install -r requirements.txt
```

### 修改配置文件

项目根目录下有个`.env.tpl`模板配置文件。将其重命名文`.env`文件，修改里面的配置项即可。

每个配置项在文件里都有详细说明，按照说明配置即可。

### 运行

```shell
python main.py
```
